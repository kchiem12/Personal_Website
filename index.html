Portfolio

SideQuest

App to find gigs; won best overall in AppDev HackChallenge

In the span of two weeks, I, along with a group of four others, this app, where users can log in and post gigs, and search for gigs to do. I mainly did the front-end, using Swift and XCode.

I used Alamofire to connect the backend in order to save user info and gigs.

Workout Tracker

A web-based application to track your workouts

Created in August 2022, the application is made with react and Firebase is used as BaaS. It authenticates the user, and maintains a calendar of the users activity/workouts. Currently transitioning the backend to MongoDB using NodeJS and Express.

Railroad Damage Detection

Detects defective railroads with machine learning

Created in August 2022. I used PyTorch in order to create a model that detects a defective railroad given an image. Through training the model, it resulted in an accuracy of 94.33% on the training set, and 77.27% on the test set.
